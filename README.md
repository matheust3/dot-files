# dot-files
My dot files

### WINDOWS
- 1- Instale o Starship
- 2- Instale a fonte "Caskaydia Cove Nerd Font Complete.ttf" e configure o terminal para usar essa fonte
- 3- Configure a variavel de configuração $STARSHIP_CONFIG  ->  "C:\Users\mathe\.config\starship.toml"
- 2 - Cole 'Microsoft.PowerShell_profile.ps1' -> $PSHOME

### Linux
- 1 - Install Zsh
- 2 - Install Oh My Zshell
- 3 - Change your default shell to zsh
- 4 - Install "zsh-autosuggestions"
- 5 - Install Starship
> Se **starship.toml** estiver em **C:\Users\mathe\.config\starship.toml** não é necessario baixar o arquivo de configuração do Starship, mas se não estiver deve baixar e mudar a variavel **STARSHIP_CONFIG** em **.zshrc**
